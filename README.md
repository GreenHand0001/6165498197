# 项目简介

## 核心功能
- 提供中草药文化知识普及与传播平台
- 展示中草药的基本信息与特性
- 支持用户学习中草药相关知识
- 提供中草药文化宣传的互动渠道
- 整理收录大量中草药相关论文资源(1w+篇)

## 适用场景
- 中草药文化知识学习与教育
- 中草药信息查询与了解
- 中草药文化宣传与推广
- 学术研究中草药资料参考
- 群众草药文化意识提升

## 技术特点
- 基于 SpringBoot 后端框架开发
- 采用 Vue 前端技术栈构建
- 整合大量学术论文资源
- 提供用户友好的信息交互界面
